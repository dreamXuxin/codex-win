# ETF监控加仓告警

这个项目用于监控 ETF 当前价格，并根据持仓成本价触发回撤加仓告警。

## 运行方式

1. 复制 `config/email_config.example.json` 为 `config/email_config.json`
2. 在 `config/email_config.json` 里填入 QQ 邮箱 SMTP 授权码
3. 双击 `run_monitor.bat` 常驻运行

监控规则：

| 时间 | 动作 |
| --- | --- |
| 工作日 09:30 | 抓取价格并检查加仓告警 |
| 工作日 10:30 | 抓取价格并检查加仓告警 |
| 工作日 11:30 | 抓取价格并检查加仓告警 |
| 工作日 13:00 | 抓取价格并检查加仓告警 |
| 工作日 14:00 | 抓取价格并检查加仓告警 |
| 工作日 15:00 | 发送今日涨跌表 |

告警邮件发送到：`1204926020@qq.com`。

同一只 ETF 同一天同一档位只发送一次；如果继续跌破更深档位，会继续发送新的告警。
